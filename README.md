# AdoraCoatings Dashboard

Business management portal for AdoraCoatings — Inventory, Lead Generation, Orders, and Reports.

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (charts)
- React Router v6

## Pages
- **Login** — Green & white branded login
- **Dashboard** — Revenue overview, top products, recent leads, alerts
- **Inventory** — Product stock management with filters and add/edit
- **Lead Generation** — Lead table + Email automation + WhatsApp automation
- **Orders** — Order tracking with status and payment info
- **Reports** — Revenue vs expense, category pie, lead conversion charts
- **Settings** — Company info, notifications, team, security

## Dev Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
